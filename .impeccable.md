## Design Context

### Users
DeFi power users bridging USDC/USDT across EVM, Cardano, and Midnight. Context: production bridge interface, not a developer tool.

### Brand Personality
Precise, trustworthy, minimal.

### Aesthetic Direction
Clean production DeFi app. No developer panels in default view. Light theme with teal accents (existing palette). No mock/demo language in production builds.

### Design Principles
1. Every label earns its place -- no redundant "demo" qualifiers
2. Progressive disclosure -- advanced tools hidden behind DEV mode
3. Trust through clarity -- real proof and finality data shown prominently
4. Chain-aware copy -- labels adapt to the actual chain/token context
